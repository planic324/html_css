animation 이슈(safari)

HOW?

  • safari에서 동작안하는 이슈를 해결해보자 !
  • 결론부터 말하자면 바로 keyframes 작성 방법!

이렇게 쓰면 safari에서 안됩니다 !

@keyframes test_circle{ 0%{transform: translateX(-300%) scale(0.3);} 40%{transform: translateX(0) scale(1);} 41%{transform: rotate(0deg);} 100%{transform: rotate(630deg);} }

safari에서 안되면 이렇게!

@keyframes test_circle{ 0%{transform: translateX(-500%) scale(0.3) rotate(0deg);} 40%{transform: translateX(0) scale(1) rotate(0deg);} 100%{transform: translateX(0) scale(1) rotate(630deg);} }
